# Phaser 3 boilerplate
> This is a template for Phaser.js game engine

#### Requirements
You only need **node.js** installed on your computer and it's ok.

#### Installation
install dependencies:
```
yarn install
```

#### Development
Run the webpack-dev-server on http://lcoalhost:8001
```
yarn start
```

#### Production
Build the app
```
yarn build
```
